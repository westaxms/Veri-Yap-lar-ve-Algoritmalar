# Insertion Sort Projesi

## Veri Seti
Verilen dizi: `[22, 27, 16, 2, 18, 6]`

### Aşamalar (Insertion Sort):
1. [22, 27, 16, 2, 18, 6]
2. [16, 22, 27, 2, 18, 6]
3. [2, 16, 22, 27, 18, 6]
4. [2, 16, 18, 22, 27, 6]
5. [2, 6, 16, 18, 22, 27]

## Big-O Gösterimi
- Best Case: O(n)
- Average Case: O(n²)
- Worst Case: O(n²)

**18 sayısı sıralandıktan sonra ortada kaldığı için → Average Case**
